Manage Report Setup

The DIY Transparency Reporting tool relies on different data categories to collate the kinds of information retained, to describe how an organization responds to government access requests, and to explain how often and with what effect it receives requests. Each of the reporting types associated with this tool are defined in the accompanying Guide [Link out to place in future]. Users can add new categories and responses for the types of reports in the ‘Manage Report Setup’ section. These categories and responses can be added when generating a report, but can only be deleted from this section.
